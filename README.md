# ag3nt-coin

Agent-native crypto for the [ADD](https://github.com/zoltankiss/agent-driven-development) economy.

See `CLAUDE.md` for context, `SPEC.html` for the design (open in a browser).

## Run

```bash
bun install
bun run dev
```

Server on `:3041` (override with `PORT`). Chain at `data/chain/*.json`, validator key at `data/node.key.json`.
